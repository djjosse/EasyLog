Easylog

Open Source Wpf Easy Log for .NET/C#

Download this project as a .zip file Download this project as a tar.gz file

Welcome to .NET EasyLog.

This solution is Open Source which provides fast and easy way to create Real Time logs for your .NET application and store those logs for future investigation. You can use the "Example" project in the solution to understand how to use EasyLog.
